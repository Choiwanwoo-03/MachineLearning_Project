# 🏢 아파트 실거래가 추정 모델 개발 프로젝트
**Apartment Real Transaction Price Estimation Model**

본 프로젝트는 머신러닝 알고리즘을 활용하여 아파트의 현재 실거래가를 정교하게 추정하고, 가격 결정에 영향을 미치는 주요 변수들을 분석하는 것을 목표로 합니다.

---

## 🎯 1. 프로젝트 목적 및 목표
* **실거래가 추정:** 공공데이터 및 입지 데이터를 결합하여 특정 시점의 아파트 매매가를 고도로 정밀하게 예측합니다.
* **데이터 최적화:** 결측치 처리, 이상치 제거 및 피처 엔지니어링을 통해 모델의 일반화 성능을 극대화합니다.
* **기술적 가이드 제공:** 데이터 수집부터 모델 평가까지 전 과정에 걸쳐 체계적인 머신러닝 파이프라인을 구축합니다.

---

## 📊 2. 데이터셋 구성
프로젝트의 데이터 아키텍처는 `데이터셋 종류.xlsx` 파일을 기준으로 설계되었습니다.
* **주요 데이터원:** 국토교통부 아파트매매 실거래 상세 자료 (공공데이터포털 API)
* **독립 변수 (Features):**
  - **단지 내부:** 전용면적, 층수, 건축년도, 총 세대 수 등 (한초아)
  - **교육 학군:** 학교 거리, 학원가 접근성 등 (병선)
  - **교통 인프라:** 지하철역·버스 정류장 거리, GTX 수혜 여부 등 (한결)
  - **생활 환경:** 편의점·대형마트·병원 거리, 공원 접근성 등 (한결)
  - **거시 정책:** 기준 금리, LTV·DSR 규제 수준, 부동산 규제 더미 등 (완우)
* **종속 변수 (Target):** 아파트 실거래 평당가 (만원/평)

## 📈 4. 모델 평가 지표
부동산 가격 예측의 특성을 고려하여 아래 지표를 통해 모델의 성능을 다각도로 검증합니다.

**회귀 모델 (가격 추정)**
* **R² (결정계수):** 모델이 분산을 설명하는 비율 (높을수록 우수)
* **MAE (Mean Absolute Error):** 실제 가격과의 절대적 차이 측정
* **MAPE (Mean Absolute Percentage Error):** 가격 규모 대비 오차 비율 (핵심 지표)
* **이상치율 (±15% 초과):** 허용 범위를 벗어난 예측 비율

**분류 모델 (가격 등급 분류)**
* **Accuracy:** 전체 정확도
* **Precision / Recall / F1 (macro):** 클래스 불균형을 고려한 성능 지표
